
#  Mobile App Store Analysis

This project analyzes Google Play Store and Apple App Store datasets to
identify profitable app profiles using data cleaning, exploratory data
analysis (EDA), and visualization techniques.

---

##  Objective

- Analyze app categories and user engagement
- Compare free vs paid app performance
- Identify high-potential app profiles
- Generate business insights from data
